# MediaFetch — Installation (End User)

## Deutsch

### Installation

1. Laden Sie `MediaFetch-win64.zip` von [GitHub Releases](https://github.com/your-org/MediaFetch/releases) herunter
2. Entpacken Sie das ZIP in einen Ordner Ihrer Wahl (z. B. `C:\Programme\MediaFetch`)
3. Starten Sie `MediaFetch.exe`
4. Bei Windows SmartScreen: **Weitere Informationen** → **Trotzdem ausführen**

### Erste Schritte

1. Öffnen Sie **Einstellungen** und wählen Sie den Download-Ordner
2. Fügen Sie Video-URL(s) ein (eine pro Zeile)
3. Klicken Sie **Zur Warteschlange hinzufügen** und wählen Sie Qualität/Format
4. Downloads erscheinen in der Warteschlange mit Fortschrittsanzeige

### Update

Wenn eine neue Version verfügbar ist, zeigt die App einen Hinweis mit Link zur Release-Seite:

1. App schließen
2. Neues ZIP herunterladen
3. Alten Installationsordner ersetzen (oder neuen Ordner verwenden)
4. App neu starten

### yt-dlp aktualisieren

In **Hilfe → Updates prüfen** können Sie yt-dlp separat aktualisieren, ohne die gesamte App zu ersetzen.

## English

### Installation

1. Download `MediaFetch-win64.zip` from [GitHub Releases](https://github.com/your-org/MediaFetch/releases)
2. Extract to any folder (e.g. `C:\Programs\MediaFetch`)
3. Run `MediaFetch.exe`
4. If Windows SmartScreen appears: **More info** → **Run anyway**

### First steps

1. Open **Settings** and choose your download folder
2. Paste video URL(s) (one per line)
3. Click **Add to queue** and select quality/format
4. Downloads appear in the queue with live progress

### Updating

When a new version is available, the app shows a notification with a link to the release page:

1. Close the app
2. Download the new ZIP
3. Replace the installation folder (or use a new folder)
4. Restart the app

### Updating yt-dlp

Use **Help → Check for updates** to update yt-dlp separately without replacing the entire app.
